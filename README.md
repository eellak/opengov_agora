# Ηλεκτρονική Πλατφόρμα για την παρακολούθηση δημόσιων συμβάσεων


Το αντικείμενο της συγκεκριμένης δράσης «Δράση 2 - Ηλεκτρονική Πλατφόρμα για την παρακολούθηση δημόσιων συμβάσεων» είναι η ανάπτυξη, παράδοση και τεκμηρίωση πρότυπης πλατφόρμας για την καταχώριση και παρακολούθηση συμβάσεων σε κεντρικό επίπεδο. Η πλατφόρμα ενσωματώνει βασικές ροές εργασίας για τη διαχείριση των συμβάσεων (αίτημα προμήθειας, αγαθού, έργου, υπηρεσίας, συμβασιοποίηση, πληρωμή) και θα δίνει τη δυνατότητα της προβολής του σχετικού περιεχομένου μέσα από δημόσιες διεπαφές.
